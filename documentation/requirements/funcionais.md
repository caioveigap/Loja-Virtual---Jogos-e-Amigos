Aqui estão os Requisitos Funcionais com o prefixo solicitado adicionado a cada descrição:

## ✅ Requisitos Funcionais

### 📧 Conta
- **RF.01:** O sistema deve permitir cadastro de usuário  
- **RF.02:** O sistema deve permitir autenticação de contas de usuário  
- **RF.03:** O sistema deve permitir autenticação de administradores com permissões adicionais
- **RF.28:** O sistema deve confirmar e-mail autenticado
- **RF.29:** O sistema deve permitir que o usuário atualize seus dados cadastrais  
- **RF.04:** O sistema deve exibir lista de seguidores do usuário (quando pública)  
- **RF.05:** O sistema deve exibir lista de perfis seguidos pelo usuário (quando pública)  
- **RF.06:** O sistema deve exibir lista de jogos adquiridos pelo usuário(quando pública)  
- **RF.07:** O sistema deve exibir lista de jogos publicados do usuário (quando pública)  

### 🔐 Privacidade do Perfil
- **RF.08:** O sistema deve permitir que o usuário altere a visibilidade da sua lista de seguidores  
- **RF.09:** O sistema deve permitir que o usuário altere a visibilidade da sua lista de perfis seguidos  
- **RF.10:** O sistema deve permitirque o usuário altere visibilidade da sua lista de jogos adquiridos  
- **RF.11:** O sistema deve permitir alterar visibilidade da sua lista de jogos publicados  

### 🛒 Loja
- **RF.12:** O sistema deve permitir que o usuário adqira jogos (gratuitos ou pagos)  
- **RF.13:** O sistema deve permitir que o usuário publique jogos  
- **RF.14:** O sistema deve permitir que o usuário baixe jogos adquiridos    

### 💳 Pagamentos
- **RF.15:** O sistema deve permitir pagamento via Pix  
- **RF.16:** O sistema deve permitir pagamento via Cartão de crédito

### 🛡️ Moderação e Interação
- **RF.17:** O sistema deve permitir que o usuário reporte jogos  
- **RF.18:** O sistema deve permitir que o administrador avalie reports
- **RF.19:** O sistema deve permitir que o usuário avalie jogos (com ou sem comentário)  
- **RF.20:** O sistema deve restringir avaliação a usuários que adquiriram o jogo  

### 🔍 Busca e Social
- **RF.21:** O sistema deve permitir busca de jogos por categoria  
- **RF.22:** O sistema deve permitir busca de jogos por nome  
- **RF.23:** O sistema deve permitir que usuários enviem solicitração de seguimento a outros
- **RF.24:** O sistema deve permitir que um usuário deixe de seguir outro usuário
- **RF.25:** O sistema deve permitir que o usuário visualize solicitações de seguimento  

### 🧑‍💻 Desenvolvedor
- **RF.26:** O sistema deve permitir que o usuário registre DevLogs

### 📊 Compras
- **RF.27:** O sistema deve permitir que o usuário visualize seu histórico de compras  
- **RF.28:** O sistema deve exibir status da compra  
- **RF.29:** O sistema deve associar jogo à conta compradora  
- **RF.30:** O sistema deve permitir que o usuário visualize a página detalhada de um jogo  

### 🛠️ Gestão de Jogos
- **RF.31:** O sistema deve permitir que o usuário edite seus jogos publicados
- **RF.32:** O sistema deve permitir que o usuário remova seus jogos publicados  
- **RF.33:** O sistema deve permitir que o administrador remova jogos

### ⚠️ Administração
- **RF.34:** O sistema deve permitir que o administrador consulte denúncias
-  **RF.35:** O sistema deve permitir que o administrador acesse as denúncias 
- **RF.36:** O sistema deve permitir que o administrador consulte contas de usuários  
- **RF.37:** O sistema deve permitir que o administrador delete a conta de um usuário

### 🔔 Notificações
- **RF.38:** O sistema deve enviar ao usuário notificações de interações relevantes
