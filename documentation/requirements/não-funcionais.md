# ⚙️ Requisitos Não-Funcionais

### 🔐 Segurança
- **RNF.01:** Confidencialidade de dados  
- **RNF.02:** Criptografia de comunicação  

### ⚡ Desempenho
- **RNF.03:** Buscas respondidas em até 2 segundos  
- **RNF.04:** Disponibilidade de 99%  
- **RNF.05:** Suporte a 500 usuários simultâneos  
- **RNF.06:** Latencia máxima de 200 ms em operações

### 📈 Escalabilidade e Dados
- **RNF.07:** Armazenar até 1TB de dados
- **RNF.08:** Preservar a integridade dos dados  

### 📜 Auditoria e Logs
- **RNF.09:** Registro de logs  
- **RNF.10:** Rastreabilidade de ações  

### 📱 Usabilidade e Manutenção
- **RNF.11:** Adaptação aos tamanhos variados de tela  
- **RNF.12:** Atualizações sem perda de dados existentes
- **RNF.13:** Tolerância a falhas, com recuperação de dados 
- **RNF.14:** Validação de entradas do usuráio, para evitar erros e maus usos
- **RNF.15:** Conformidade legal no tratamento de dados dos usuários
